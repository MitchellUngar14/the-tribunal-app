# Ratabase Web UI Sandbox Landing Page - Context

This file provides context for the Ratabase Web UI Sandbox landing page.

## File Structure

The sandbox environment assets are located in the `Sandbox/` directory.
- `Sandbox/index.html`: The main landing page file.
- `Sandbox/css/main.css`: Custom styles for the landing page.
- `Sandbox/js/main.js`: Custom scripts for the landing page.
