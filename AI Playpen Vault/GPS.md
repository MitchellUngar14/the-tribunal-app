# Sub-Project: GPS

This file contains context for the GPS sub-project.

---

## Role Definition

A GPS (Guide, Plan, Support) is a manager role responsible for guiding and supporting a team of employees. Key responsibilities include:
- Providing resources and support to team members.
- Conducting employee annual reviews.

---

## Employee Context Files

Employee-specific context is stored in individual notes within the 'GPS/' folder. These files serve as a record for notes about employees and should include timestamps for all entries.
