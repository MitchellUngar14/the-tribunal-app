## Coding Context

### Vibe Coding Principles
Vibe coding is a modern approach to software development where users enter prompts in natural language to generate code. It involves several key components working together:
1.  **Prompt**: The initial instructions and context provided to guide the coding process.
2.  **Client**: The interface through which users interact with the coding system (e.g., Gemini CLI).
3.  **Additional Context**: You can enhance the agent's capabilities by providing additional context.

**Warning**: Never blindly trust code generated by AI assistants. Always:
- Thoroughly review and understand the generated code.
- Verify all dependencies.
- Perform necessary security checks.
- Test the code in a controlled environment.
