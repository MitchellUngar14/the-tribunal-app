# Project: Tribunal

This project is designed to simulate a team of specialized AI agents that collaborate to answer coding questions and solve complex software engineering problems.

---

## The Team

This project consists of several AI agents, each with a unique specialty. The goal is to provide a comprehensive and well-rounded answer by having each agent contribute its expertise.

### Agents

- [[Tribunal/General Project Manager|General Project Manager]]: Coordinates the other agents, breaks down the user's request, and synthesizes the final answer.
- [[Tribunal/Python Specialist|Python Specialist]]: Expert in Python, data science, and backend logic.
- [[Tribunal/Frontend Specialist|Frontend Specialist]]: Expert in HTML, CSS, JavaScript, and modern frontend frameworks like React.
- [[Tribunal/Database Specialist|Database Specialist]]: Expert in SQL, database design, and performance optimization.
- [[Tribunal/Java Specialist|Java Specialist]]
- [[Tribunal/Java Spring Specialist|Java/Spring Specialist]]
- [[Tribunal/Javascript Specialist|Javascript Specialist]]
- [[Tribunal/React Specialist|React Specialist]]
- [[Tribunal/Postgres Specialist|Postgres Specialist]]
- [[Tribunal/Postgresql Specialist|Postgresql Specialist]]
- [[Tribunal/DynamoDB Specialist|DynamoDB Specialist]]
- [[Tribunal/MSSQL Specialist|MSSQL Specialist]]

---

## Workflow

1.  The user provides a coding question or problem.
2.  The **General Project Manager** analyzes the request and determines which specialists are needed.
3.  The relevant specialists are "consulted" (i.e., I will use their context files to generate their responses).
4.  The specialists provide their individual contributions.
5.  The **General Project Manager** synthesizes these contributions into a final, coherent answer for the user.
