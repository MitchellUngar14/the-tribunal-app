### **Phase 1: Core Frontend Component (React)**

Our Frontend Specialist has designed a complete, reusable stopwatch component using React. This approach provides a modern, interactive user interface that manages the stopwatch's state (time, laps, running status) entirely within the user's browser.

#### **Key Features & Logic:**

*   **State Management:** Uses React Hooks (`useState`, `useRef`, `useEffect`) to efficiently manage the timer's state without causing unnecessary re-renders.
    *   `useState` tracks the elapsed time, the list of laps, and whether the timer is currently running.
    *   `useRef` holds the interval ID, preventing re-renders every time the interval is set or cleared.
    *   `useEffect` manages the `setInterval` lifecycle, ensuring the timer is correctly started and stopped, and cleaned up when the component is removed from the screen.
*   **Core Functions:** Implements `start`, `stop`, `lap`, and `reset` logic.
*   **Time Formatting:** Includes a helper function to convert milliseconds into a human-readable `HH:MM:SS.ms` format.

#### **File Structure:**

```

|   |-- Stopwatch.js      # The main component logic
```

#### **Component Code: `Stopwatch.js`**

```javascript
import React, 'react';
import './Stopwatch.css';

const Stopwatch = () => {
    const [time, setTime] = React.useState(0);
    const [laps, setLaps] = React.useState([]);
    const [isRunning, setIsRunning] = React.useState(false);
    const timerRef = React.useRef(null);

    React.useEffect(() => {
        if (isRunning) {
            timerRef.current = setInterval(() => {
                setTime(prevTime => prevTime + 10);
            }, 10);
        } else {
            clearInterval(timerRef.current);
        }
        return () => clearInterval(timerRef.current); // Cleanup on unmount
    }, [isRunning]);

    const handleStartStop = () => {
        setIsRunning(!isRunning);
    };

    const handleLap = () => {
        if (isRunning) {
            setLaps([...laps, time]);
        }
    };

    const handleReset = () => {
        setIsRunning(false);
        setTime(0);
        setLaps([]);
    };

    const formatTime = (timeInMs) => {
        const milliseconds = `0${(timeInMs % 1000) / 10}`.slice(-2);
        const seconds = `0${Math.floor(timeInMs / 1000) % 60}`.slice(-2);
        const minutes = `0${Math.floor(timeInMs / 60000) % 60}`.slice(-2);
        return `${minutes}:${seconds}.${milliseconds}`;
    };

    return (
        <div className="stopwatch-container">
            <div className="timer-display">{formatTime(time)}</div>
            <div className="button-controls">
                <button onClick={handleStartStop}>{isRunning ? 'Stop' : 'Start'}</button>
                <button onClick={handleLap} disabled={!isRunning}>Lap</button>
                <button onClick={handleReset}>Reset</button>
            </div>
            <ul className="lap-list">
                {laps.map((lap, index) => (
                    <li key={index}>
                        Lap {index + 1}: {formatTime(lap)}
                    </li>
                ))}
            </ul>
        </div>
    );
};

export default Stopwatch;
```

### **Phase 2: Backend & Data Persistence (Node.js & SQLite)**

Our Backend Specialist proposes a lightweight yet robust solution using a Node.js server with the Express framework and an SQLite database. This stack is ideal for rapid development and easy setup.

#### **Architecture Overview:**

*   **API Server:** An Express.js server will expose API endpoints for the frontend to communicate with.
*   **Database:** SQLite will be used for its simplicity (it's just a file), making it perfect for this project without requiring a separate database server.
*   **Data Model:** A single `sessions` table will store the final time and all associated laps for each stopwatch run.

#### **Data Model (`sessions` table):**

*   `id` (INTEGER, PRIMARY KEY, AUTOINCREMENT)
*   `total_time` (INTEGER) - The final time in milliseconds.
*   `laps` (TEXT) - A JSON string containing an array of lap times in milliseconds.
*   `created_at` (DATETIME) - The timestamp when the session was saved.

#### **API Endpoints:**

1.  **`POST /api/sessions`**: Saves a completed stopwatch session.
    *   **Request Body:** `{ "total_time": 123450, "laps": [30120, 65430, 123450] }`
    *   **Response:** `{ "message": "Session saved successfully", "id": 1 }`
2.  **`GET /api/sessions`**: Retrieves a history of all saved sessions.
    *   **Response:** `[ { "id": 1, "total_time": 123450, "laps": [...], "created_at": "..." }, ... ]`

#### **Example Backend Code: `server.js`**

```javascript
const express = require('express');
const sqlite3 = require('sqlite3').verbose();
const cors = require('cors');

const app = express();
const PORT = 3001;

app.use(cors());
app.use(express.json());

// Initialize SQLite database
const db = new sqlite3.Database('./stopwatch.db', (err) => {
    if (err) {
        console.error(err.message);
    }
    console.log('Connected to the stopwatch database.');
    // Create sessions table if it doesn't exist
    db.run(`CREATE TABLE IF NOT EXISTS sessions (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        total_time INTEGER NOT NULL,
        laps TEXT,
        created_at DATETIME DEFAULT CURRENT_TIMESTAMP
    )`);
});

// POST endpoint to save a session
app.post('/api/sessions', (req, res) => {
    const { total_time, laps } = req.body;
    const lapsJson = JSON.stringify(laps);
    const sql = `INSERT INTO sessions (total_time, laps) VALUES (?, ?)`;
    
    db.run(sql, [total_time, lapsJson], function(err) {
        if (err) {
            return res.status(400).json({ "error": err.message });
        }
        res.json({
            "message": "Session saved successfully",
            "id": this.lastID
        });
    });
});

// GET endpoint to retrieve all sessions
app.get('/api/sessions', (req, res) => {
    const sql = "SELECT * FROM sessions ORDER BY created_at DESC";
    db.all(sql, [], (err, rows) => {
        if (err) {
            return res.status(400).json({ "error": err.message });
        }
        // Parse the laps JSON string back into an array
        const results = rows.map(row => ({
            ...row,
            laps: JSON.parse(row.laps || '[]')
        }));
        res.json(results);
    });
});

app.listen(PORT, () => {
    console.log(`Server is running on http://localhost:${PORT}`);
});
```
